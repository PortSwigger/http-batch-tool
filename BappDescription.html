HTTP Batch Tool enables efficient bulk HTTP request processing by allowing users to paste multiple URLs and send them concurrently. The extension provides a dedicated tab with an intuitive interface that displays results in a sortable table with detailed request and response viewers, streamlining reconnaissance and testing workflows.

Features

  • Concurrent processing of multiple URLs
  • Interactive results table showing URL, status code, response length, and request duration
  • Integrated request and response viewers with full message details
  • Real-time progress tracking with completion status updates
  • Right-click context menu for convenient URL pasting from clipboard
  • Automatic error handling for malformed URLs and failed requests
  • Custom User-Agent header configuration for consistent request profiling

Usage

  1. Navigate to the "HTTP Batch Tool" tab within Burp Suite
  2. Paste or type multiple URLs in the text area, one per line
  3. Alternatively, right-click in the text area and select "Paste URLs" to import from clipboard
  4. Click the "Send Requests" button to dispatch all URLs concurrently
  5. Monitor real-time progress updates in the status label
  6. Click any row in the results table to view the complete request and response details
  7. Sort results by any column to organize findings effectively
