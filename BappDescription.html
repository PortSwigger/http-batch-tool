HTTP Batch Tool enables bulk HTTP request processing by allowing users to paste multiple URLs and send them all through Burp's proxy with a single click. This extension streamlines reconnaissance and testing workflows by automating the dispatch of numerous requests for analysis and inspection.

Features

  • Batch processing of multiple URLs from a text input area
  • Multithreaded request handling for improved performance
  • Custom User-Agent header injection for consistent request formatting
  • Real-time output and error reporting in the Extender console
  • Support for both HTTP and HTTPS protocols with automatic port detection

Usage

  1. Navigate to the HTTP Batch Tool tab within Burp Suite
  2. Paste or type multiple URLs in the text area, one per line
  3. Alternatively, right-click in the text area and select Paste URLs to import from clipboard
  4. Click Send Requests to dispatch all URLs
  5. Review captured requests and responses in Burp's Logger tab for further analysis
